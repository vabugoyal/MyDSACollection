What is a packaged task in cpp ?

A wrapper around a callable (function/lambda) that lets 
you run it later and get its result via a std::future.

packaged_task = function + promise (hidden inside)


When to use
    Use when you want control over when/where the function runs
    Prefer over std::async when:
        you have your own thread pool
        you need manual scheduling

int main() {
    // launch async task
    std::future<std::string> fut = std::async(std::launch::async, fetch_data);

    std::cout << "Doing other work...\n";

    // later: get result (blocks only here if not ready)
    std::string result = fut.get();

    std::cout << "Result: " << result << "\n";
